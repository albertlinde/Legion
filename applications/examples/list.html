Legion List Examples

This page contains examples of using the Legion framework.

Usage

    var legion = new Legion();
    legion.join();
    var objectStore = legion.getObjectStore();

    var list_1 = objectStore.get("list_1", List.STATE_List);
    var list_2 = objectStore.get("list_2", List.OPERATIONS_List);
    var list_3 = objectStore.get("list_3", List.DELTA_List);

    list_*.add(pos, value);
    list_*.remove(pos);
    list_*.get(pos)

    console.log(list_*.asArray()); //Array
    console.log(list_*.length()); //Number
